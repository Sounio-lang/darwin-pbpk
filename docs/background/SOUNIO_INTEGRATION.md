## Overview

MedLang can now compile to **Sounio** (L0 epistemic systems language) in addition to Stan and Julia. This integration provides:

- **Compile-time dimensional analysis** - Unit errors caught at compile time
- **Epistemic computing** - Uncertainty and confidence tracking through all computations
- **Algebraic effects** - Safe mutation, IO, and GPU operations
- **Refinement types** - Value constraints verified by SMT solver

## What Was Implemented

### 1. Sounio Code Generator (`Darwin-medlang/compiler/src/codegen/sounio.rs`)

**Features**:
- ✅ **Unit type system**: Automatically generates Sounio unit types from MedLang dimensions
- ✅ **Struct generation**: Creates `Params`, `State`, `StateDerivatives`, `Results` structs
- ✅ **ODE system**: Generates `ode_system()` function with algebraic effects
- ✅ **ODE solver**: Euler integration (will upgrade to Tsit5 when darwin.pbpk stdlib exists)
- ✅ **Expression translation**: Full IR expression → Sounio expression mapping
- ✅ **Main entry point**: Generates runnable `main()` function

**Example transformation**:

```medlang
// MedLang input
model OneCompOral {
    state A_gut: DoseMass
    param Ka: RateConst

    dA_gut/dt = -Ka * A_gut
    obs C_plasma: ConcMass = A_central / V
}
```

↓ **Compiles to** ↓

```d
// Sounio output
module OneCompOral

import std.math.{exp, log, sqrt}
import std.effects.{Mut, IO, Alloc}

type RateConst = 1 / Time<h>
type DoseMass = Mass<mg>
type ConcMass = Mass<mg> / Volume<L>

struct Params {
    Ka: RateConst,
}

struct State {
    A_gut: DoseMass,
}

fn ode_system(state: &State, params: &Params, t: h) -> StateDerivatives {
    let A_gut = state.A_gut
    let Ka = params.Ka

    StateDerivatives {
        d_A_gut: (-Ka * A_gut),
    }
}

fn main() -> effect[IO, Alloc, Mut] {
    // ... simulation code
}
```

## Key Benefits

### 1. **Compile-Time Unit Safety**

MedLang dimension annotations → Sounio unit types:

| MedLang Dimension | Sounio Type | Example |
|-------------------|----------------|---------|
| `Mass` | `DoseMass = Mass<mg>` | Drug amount |
| `Volume` | `Volume<L>` | Distribution volume |
| `Clearance` | `Clearance = Volume<L> / Time<h>` | Drug clearance |
| `RateConst` | `RateConst = 1 / Time<h>` | Absorption rate |
| `Concentration` | `ConcMass = Mass<mg> / Volume<L>` | Plasma concentration |
| `Fraction` | `Fraction = { x: f64 \| 0.0 <= x && x <= 1.0 }` | Bioavailability |

**Errors caught at compile time**:
```d
let CL: Clearance = 10.0        // OK
let wrong: Volume = CL           // ERROR! Clearance != Volume
let bad = CL + V                 // ERROR! Can't add Clearance + Volume
```

### 2. **Epistemic Computing**

Every value can carry uncertainty and confidence:

```d
let dose: EpistemicValue<mg> = EpistemicValue::new(
    value: 100.0,
    uncertainty: 5.0,      // ±5 mg
    confidence: 0.95,       // 95% confidence
    source: Source::Measurement
)

// Uncertainty propagates through computations (GUM-compliant)
let conc = dose / volume
assert(conc.confidence <= dose.confidence)  // Monotone non-increasing
```

### 3. **Algebraic Effects**

Safe mutation, IO, and GPU operations:

```d
fn simulate_pbpk(params: Params) -> effect[IO, Mut, Alloc] Vec<(h, uM)> {
    var state = State::new()

    // Mutation tracked by effect system
    state.A_gut = 100.0  // effect[Mut]

    // IO tracked
    println!("Starting simulation")  // effect[IO]

    // Allocation tracked
    let results = Vec::new()  // effect[Alloc]

    results
}
```

### 4. **Provenance Tracking**

```d
// Every value knows where it came from
let Ka: RateConst with Provenance = from_literature(
    value: 1.5,
    cv_percent: 30.0,
    source: "Smith et al. 2023"
)
```

### Phase 2: Julia ↔ Sounio Bridge

```julia
# julia-migration/src/DarwinPBPK/sounio/Sounio.jl

module Sounio

export compile_sounio, run_sounio, EpistemicValue

"""
Call Sounio compiler (dc) from Julia
"""
function compile_sounio(source_file::String; release=true)
    binary = replace(source_file, ".sio" => "")
    flags = release ? "--release" : ""
    run(`souc build $source_file $flags -o $binary`)
    return binary
end

"""
Epistemic value type matching Sounio
"""
struct EpistemicValue{T}
    value::T
    uncertainty::Float64
    confidence::Float64
    source::Symbol
end

end # module
```
